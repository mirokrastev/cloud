# cloud-api

## Requirements
Python >= 3.8</br>
```pip install -r requirements.txt```</br>
Configure ```.env``` from ```env_example``` template

## Features
Cloud storage, built with DRF. It consists of 4 different types of users, monthly subscriptions, etc.

## Tests
To run tests, use ```pytest```
